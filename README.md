# make_python_CLI_board

> Python CLI 게시판을 단계적으로 구현하며 Python 문법과 자료구조, CRUD, 객체지향, Git/GitHub 사용 습관을 학습하는 프로젝트

---

## 📖 Project Overview

이 프로젝트는 실제 서비스를 만드는 것이 아니라 Python을 이용하여 CLI 게시판을 구현하는 과정에서 개발의 기초를 학습하기 위한 프로젝트입니다.

기능을 하나씩 추가하면서 CRUD, 객체지향, 회원관리, 권한관리까지 단계적으로 구현합니다.

---

## 🎯 Learning Goals

- Python 문법을 실제 코드에 적용
- 자료구조(List, Dictionary, Class) 활용
- CRUD 구현 과정 이해
- 객체지향 프로그래밍 기초 학습
- Git & GitHub 사용 습관 형성

---

## 🗺 Roadmap

- Phase 1. CLI와 게시글 CRUD
- Phase 2. 코드 리팩터링과 객체지향
- Phase 3. 회원가입과 로그인
- Phase 4. 권한 관리
- Phase 5. 데이터베이스 통합

자세한 내용은 `docs/project-plan.md`에서 확인할 수 있습니다.
